# ☕ Köşe Kafe — Web Sitesi

Bir kafe için tasarlanmış çok sayfalı tanıtım sitesi. 

---

## 📄 Sayfalar

| Dosya | Sayfa | Layout Tipi |
|---|---|---|
| `index.html` | Ana Sayfa | Hero + Kart şeridi |
| `menu.html` | Menü | Flex kart grid |
| `hakkimizda.html` | Hakkımızda | 2 sütun layout |
| `rezervasyon.html` | Rezervasyon | Sidebar (form + bilgi) |
| `iletisim.html` | İletişim | Sidebar (bilgi + form) |
| `style.css` | — | Tüm sayfalara ait stiller |

---

 

## 🎨 Tasarım

- **Renk paleti:** Krem, kahverengi ve altın tonları
- **Fontlar:** [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) + [Jost](https://fonts.google.com/specimen/Jost) (Google Fonts)
- **Tema:** Sıcak, samimi, butik kafe atmosferi

---

## 🚀 Kurulum

1. Repoyu klonla veya ZIP olarak indir
2. Klasörü aç
3. `index.html` dosyasını tarayıcıda çalıştır

 

İnternet bağlantısı gereklidir (Google Fonts için).

---

## 📁 Klasör Yapısı

```
kafe/
├── index.html
├── menu.html
├── hakkimizda.html
├── rezervasyon.html
├── iletisim.html

```
